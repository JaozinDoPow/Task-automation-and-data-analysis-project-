# Task-automation-and-data-analysis-project-
This is a Python program I created to improve my skills, without any real technical purpose. It performs an automation and data analysis process simulating the sending of a report to a sales company, using pyautogui and pandas. It does the following: Opens the browser -> Accesses a website that simulates a company database (Google Drive) -> Accesses the folder and file containing the data -> Downloads the file -> Accesses the data in the file -> Calculates how good the sales were based on an average, and also calculates the total revenue and number of sales -> Sends an email with the analysis of all the data, in an organized and structured way.Obviously, I organized it in a way that suited my computer and situation, but it can be reorganized to access any type of system or work on any operating system.
# Technologies
Python 3.14
PyAutoGUI
Pandas
